# Passwords and Browser Support

Many day-to-day support issues are caused by login problems, outdated sessions, or unsupported browser behavior.

## Password issues

If a user cannot sign in:

- confirm they are using the correct login page
- check whether Caps Lock is on
- ask them to retry with a fresh password reset when allowed by clinic policy
- verify that the account has not been disabled

## Browser issues

If the site loads incorrectly:

- refresh the page
- sign out and sign back in
- try a supported modern browser
- clear cached session data if your clinic support policy allows it

## When to involve an administrator

Escalate when:

- multiple users are affected at once
- a specific page repeatedly fails to load
- a user keeps getting locked out
- changes do not save consistently
