# Appointments Report

The `Appointment Report` helps clinic teams review visit volume, check-ins, completions, cancellations, types, and room usage.

## What users can do here

- select a date range
- review appointment metrics
- compare status and type breakdowns
- inspect the detailed appointment table

## Key metrics on this page

The report can show:

- total appointments
- scheduled
- checked in
- completed
- cancelled

## Detailed sections

The page may include:

- status breakdown
- appointment type breakdown
- appointment list with date, time, parent, pet, staff, room, and department

## When to use this report

Use this report when:

- reviewing front-desk performance
- checking cancellation patterns
- understanding room or department load
- preparing staffing plans
