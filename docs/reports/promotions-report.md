# Promotions Report

The `Promotions Report` focuses on coupon usage, membership savings, renewal reminders, and promotion-related refunds.

## What users can do here

- review promotion performance by date range
- export promotion data
- check coupons that need attention
- review membership renewals due
- use `Queue Renewal Reminders` when allowed

## Key metrics on this page

The report can show:

- promoted invoices
- coupon savings
- membership savings
- total savings
- refunded savings
- net savings
- coupon uses
- member uses
- renewals due
- membership sales

## Main sections

The page may include:

- daily savings trend
- coupons needing attention
- recent membership sales
- recent promotion refunds
- membership renewals due
- top coupons
- membership savings by plan
- recent promoted invoices

## Best use cases

Use this report when:

- reviewing the impact of discounts and memberships
- checking whether a coupon should be paused
- preparing follow-up for expiring memberships
