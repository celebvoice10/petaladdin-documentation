# Clinical Report

The `Clinical Report` summarizes consultations, visit types, completion status, diagnoses, and follow-up load for the selected period.

## What users can do here

- select a date range
- review consultation metrics
- compare status and visit-type trends
- inspect consultation-level details

## Key metrics on this page

The report can show:

- total consultations
- completed consultations
- in-progress consultations
- follow-up count

## Detailed sections

The page may include:

- consultation status breakdown
- visit type breakdown
- detailed table with date, parent, pet, visit type, status, doctor, diagnosis, and follow-up date

## When to use this report

Use this report when:

- reviewing clinical workload
- checking follow-up volume
- spotting unfinished consultations
- monitoring diagnosis patterns over time
