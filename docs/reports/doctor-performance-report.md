# Staff Performance

The `Staff Performance` report compares consultations, completion rates, and invoice revenue across clinical staff.

## What users can do here

- review performance by date range
- export the report
- compare doctor workloads
- identify unassigned consultations

## What the report shows

Each doctor row can include:

- doctor name
- qualification
- consultation count
- completed consultation count
- completion percentage
- revenue total

The page also highlights the number of unassigned consultations in the selected period.

## When to use this report

Use this report when:

- checking doctor utilization
- reviewing completion quality
- balancing case load between doctors
- comparing clinical activity with billing output
