# Pet Parents

The `Pet Parents` module stores owner contact details and links each owner to their pets. Front-desk and admin teams use this section to keep client information accurate.

## What you can do here

- search pet parents by name, phone, or email
- filter by status
- sort by newest, oldest, name, or pet count
- open a pet parent's details page
- add a new pet parent
- edit an existing pet parent

## Main screen layout

The screen includes:

- a search bar for name, phone, and email
- filter controls for status and sort order
- a results table showing:
  - name
  - clinic
  - contact
  - number of pets
  - actions

## Adding a pet parent

1. Open `Pet Parents`.
2. Select `Add Pet Parent`.
3. Enter the owner's name and contact details.
4. Save the record.

If memberships are used in your clinic, staff may also move into membership-related tasks from this module.

## Editing a pet parent

1. Search for the owner.
2. Open the record from the list.
3. Update contact or status details.
4. Save the changes.

## Using filters well

Use search when you know the name, phone, or email. Use filters when:

- you need only active or inactive records
- you want to find the newest recently added owners
- you want to identify owners with the most linked pets

## Common mistakes to avoid

- creating duplicate pet parent records because of alternate phone numbers or spelling
- leaving email blank when the clinic uses email reminders
- editing the wrong owner because search results were not checked carefully

## Best practice

Confirm owner phone and email during every visit. Small updates at check-in help prevent follow-up and billing issues later.
