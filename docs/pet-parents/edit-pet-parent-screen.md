# Edit Pet Parent Screen

The `Edit Pet Parent` screen updates an existing owner record.

## What users can update

Staff can update:

- clinic assignment when allowed
- status
- full name
- phone
- email
- password for linked login
- membership plan and dates
- address
- notes

## Linked login behavior

When the pet parent already has a linked login:

- the linked email is shown in the password help text
- leaving the password blank keeps the current password
- entering a new password resets access for that owner

## Best practice

Update phone, email, and membership dates carefully because these affect reminders, login recovery, and billing entitlements.
