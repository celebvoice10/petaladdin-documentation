# Quotations

The `Quotations` module helps clinics prepare service or treatment estimates before checkout.

## What users can do here

- use `New Quotation`
- review quotation history
- check quotation status
- open a quotation in POS
- edit a quotation before conversion

## Main actions

The page can show:

- `POS`
- `New Quotation`
- `Open in POS`
- `Save Quotation`

## What the quotation list shows

Each quotation can display:

- quotation number
- quotation date
- validity date
- pet parent
- linked pet
- total amount
- status

## Converting a quotation to billing

When a quotation is ready for checkout:

1. Open the quotation list.
2. Select `POS`.
3. Review the loaded quotation in POS.
4. Complete the sale.
