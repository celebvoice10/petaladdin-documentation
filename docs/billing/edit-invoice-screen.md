# Edit Invoice Screen

The `Edit Invoice` screen uses the same structure as the create form, but it updates an existing invoice.

## When to use this screen

Open it when:

- the wrong pet or consultation was linked
- a line item needs correction
- payment mode or paid amount must be updated
- notes or coupon code must be changed

## Best practice

Review line discounts and tax percentages carefully before saving, especially if the invoice is already part-paid or linked to promotions.
