# Invoice Print Screen

The `Invoice Print` view is the printable version of the invoice used for clients and clinic records.

## What the print view includes

- clinic name and contact details
- invoice number
- invoice and due dates
- bill-to details
- pet and consultation details
- item table
- batch references for stock-linked lines
- subtotal
- discounts
- membership and coupon impact
- tax
- round-off values
- paid amount
- balance
- tax summary
- applied promotions
- membership purchase details
- notes

## Main actions

The print view can show:

- `Print`
- `Back`

## When to use this screen

Use it when:

- giving the client a printed invoice
- reviewing the final printable layout
- printing a billing record for clinic filing
