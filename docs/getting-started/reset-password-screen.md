# Reset Password Screen

The `Reset Password` screen is used after opening the reset link from your email.

## What the screen includes

- email field
- new password field
- confirm password field
- hidden reset token from the email link
- `Reset Password` button

## How to complete a password reset

1. Open the reset link from the email.
2. Confirm the email address is correct.
3. Enter the new password.
4. Re-enter the same password in confirmation.
5. Select `Reset Password`.

## Good password habits

- use a unique password for Petaladdin
- avoid sharing passwords across reception and billing staff
- update saved passwords on clinic browsers after a reset
