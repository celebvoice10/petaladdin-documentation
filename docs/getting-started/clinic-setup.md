# Clinic Setup

Clinic setup is usually completed by an administrator before the wider team starts using Petaladdin.

## Recommended setup order

1. Confirm clinic profile details.
2. Add staff members and assign roles.
3. Review services, pricing, and billing defaults.
4. Verify any required templates or operational settings.
5. Test a sample appointment and invoice workflow.

## Core setup areas

### Clinic profile

Record the clinic or hospital name, contact details, address, and any branding details required for printed or exported documents.

### Staff and access

Create accounts for receptionists, veterinarians, nurses, and administrators. Give each user only the access they need to perform their role.

### Services and pricing

Set up consultation types, procedures, and billing defaults so front-desk staff can use `Create Invoice` accurately.

### Final checks

Before going live:

- test a login for each staff role
- create a sample patient profile
- book a sample appointment
- generate a sample invoice
