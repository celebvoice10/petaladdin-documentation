# Login Screen

The `Login` screen is the normal entry point for Petaladdin staff accounts.

## What the screen includes

- email field
- password field
- `Remember me` checkbox
- `Forgot password?` link
- `Login` button

## How to sign in

1. Open the Petaladdin login page.
2. Enter your email address.
3. Enter your password.
4. Optional: enable `Remember me` on a trusted device.
5. Select `Login`.

## When to use `Remember me`

Use it only on a clinic-owned or trusted personal device. Avoid it on shared reception desktops unless your clinic allows it.

## Common issues

- using the wrong email address
- entering an old password after a recent reset
- trying to sign in to an inactive staff account
