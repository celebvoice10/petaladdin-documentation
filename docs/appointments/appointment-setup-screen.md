# Appointment Setup Screen

The `Appointment Setup` page lets the clinic define departments and rooms used by the scheduling board.

## What users can do here

- add departments
- edit department names, colors, and status
- add rooms
- attach rooms to departments
- activate or deactivate rooms and departments

## Department setup

Each department record can include:

- name
- color code
- status
- linked appointment count

## Room setup

Each room record can include:

- room name
- linked department
- status
- linked appointment count

## Why this matters

Accurate setup helps Petaladdin:

- group appointments by department
- schedule safely by room
- support lane-based views in the calendar
