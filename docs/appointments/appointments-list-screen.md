# Appointments List Screen

The `Appointments` screen is Petaladdin's main calendar and scheduling board.

![Appointments list screen with callouts](/img/screenshots/appointments-list-screen-annotated.jpg)

Quick guide:

- `1` opens appointment setup and scheduling actions
- `2` controls view, date, and lane layout
- `3` opens advanced filters and reset
- `4` adds an appointment directly from a day column

## What users can do here

- switch between `Day`, `Week`, and `Month` views
- change the focus date
- view lanes by `Timeline`, `Doctor`, `Room`, or `Department`
- filter by doctor, room, and department
- open appointment details
- check in eligible appointments
- drag and drop appointments to reschedule when permitted

## Main actions

The page can show:

- `Setup`
- `Schedule`
- `Filters`
- `Reset`

## Calendar behavior

- `Month` view shows appointment counts for each day.
- `Timeline` view shows appointments grouped by day.
- lane views organize appointments by doctor, room, or department.
- drag-and-drop rescheduling updates date, time, or lane depending on the current view.

## Quick actions on cards

Appointment cards can show:

- start and end time
- pet or pet parent
- doctor
- room
- department
- status badge
- `Check In` button for scheduled or confirmed visits

## Best practice

Use lane filters when the clinic is busy. Doctor and room lanes make conflict checking much easier than scanning the full board.
