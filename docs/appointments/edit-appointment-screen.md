# Edit Appointment Screen

The `Edit Appointment` screen uses the same structure as the create form, but it updates an existing visit.

## When to use this screen

Open the edit screen when you need to change:

- appointment date or time
- appointment type
- status
- doctor
- room or department
- notes
- cancellation reason

## Important use cases

Editing is most useful for:

- correcting scheduling mistakes
- updating a confirmed visit
- recording a cancellation reason
- changing the doctor or room assignment

## Best practice

If only the time or lane needs to change, the calendar drag-and-drop reschedule action may be faster. Use the full edit form when multiple fields need review.
