# Create Pet Screen

The `Add Pet` form is used to register a new pet under an existing pet parent.

![Create pet screen](/img/screenshots/create-pet-screen.jpg)

## What the form includes

- clinic
- pet parent
- status
- pet name
- species
- breed
- sex
- age
- date of birth
- color
- blood group
- microchip number
- kennel number
- spayed or neutered
- notes

## Species and breed behavior

The breed list updates based on the selected species so staff only see relevant breed options.

## Age and date of birth behavior

Petaladdin can keep `Age` and `Date of Birth` in sync:

- entering a date of birth can calculate age
- entering an age like `1Y 1M` can backfill the date of birth

## When to use this screen

Use it when:

- a newly registered pet parent adds their first pet
- a clinic branch is migrating historical pet profiles
- a second or third pet must be linked to the same owner
