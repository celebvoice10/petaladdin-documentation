# Edit Pet Screen

The `Edit Pet` screen updates an existing pet record.

## What users can update

Staff can update:

- pet parent link
- status
- pet name
- species
- breed
- sex
- age or date of birth
- color
- blood group
- microchip number
- kennel number
- spay or neuter status
- notes

## Best practice

Review species, breed, sex, and age carefully before saving. These fields affect clinical history, certificates, and preventive care workflows.
