# Pets

The `Pets` module stores each pet's clinical identity record. It connects the pet to its owner, clinic, species, breed, and status.

## What you can do here

- search pets by pet name, owner, microchip, or kennel number
- filter by species and status
- sort pet records
- open a pet's details page
- add a new pet
- edit an existing pet

## Main screen layout

The pets list shows:

- pet name
- parent name
- clinic
- profile summary such as species, breed, and sex
- actions

## Adding a pet

1. Open `Pets`.
2. Select `Add Pet`.
3. Enter the pet's identifying details.
4. Link the correct pet parent.
5. Save the record.

## Editing a pet

1. Search for the pet.
2. Open the record.
3. Update the required identity or status details.
4. Save the changes.

## Using search and filters

Search works well when you know:

- the pet's name
- the owner name
- the microchip number
- the kennel number

Use filters when you want to narrow the list by:

- species
- status
- sort order

## Why this page matters

This list is often the fastest place to confirm whether a pet already exists before booking an appointment or starting a consultation.

## Common mistakes to avoid

- creating a second record for the same pet
- linking the pet to the wrong owner
- leaving species or breed incomplete when the clinic needs those fields for reporting
