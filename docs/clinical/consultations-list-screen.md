# Consultations List Screen

The `Consultations` list screen is the main clinical register for the active clinic.

## What users can do here

- review consultation records
- open consultation details
- open the edit screen
- create a new consultation
- open prescription templates

## What the list shows

Each row can include:

- consultation date
- time
- pet
- visit type
- pet parent
- doctor
- status

## Main actions

The page can show:

- `Prescription Templates`
- `Add Consultation`

## When to use this screen

Open it when:

- reviewing the day's consultations
- finding an earlier clinical note
- starting a manual consultation not created from an appointment
