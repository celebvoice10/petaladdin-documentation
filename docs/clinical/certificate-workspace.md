# Certificates

The `Certificates` workspace helps clinics prepare printable certificates for a selected pet in the active clinic.

## What this workspace supports

Depending on permissions and clinic use, the workspace can generate:

- health certificates
- euthanasia consent forms
- surgery certificates
- vaccination certificates
- microchip certificates
- travel certificates
- spayed or neuter certificates

## How the workspace works

1. Search for the pet parent.
2. Select the correct pet.
3. Review the loaded owner and pet details.
4. Choose the certificate type.
5. Fill the extra fields required for that certificate.
6. Generate the printable output.

## What the page shows

The workspace can display:

- pet parent details
- selected pet details
- latest consultation summary
- certificate type cards
- extra form fields by certificate type
- recent vaccine suggestions

## Best practice

Check the selected pet and certificate type carefully before generating the document. The workspace reuses live pet data, so a wrong selection can carry into the certificate.
