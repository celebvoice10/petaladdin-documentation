# Prescription Templates Screen

The `Prescription Templates` area lets clinics save reusable medication plans and apply them inside consultations.

![Prescription templates list screen](/img/screenshots/prescription-templates-screen.jpg)

## What users can do here

- review existing templates
- create a new template
- edit a template
- activate or deactivate a template

## Template list screen

The list screen can show:

- template name
- scope
- item count
- status
- actions

Each template can be scoped by:

- species
- visit type

## Template form

The template form can include:

- template name
- species
- visit type
- diagnosis hint
- status

It also includes reusable medicine rows with:

- medicine type
- medicine name
- dose
- route
- duration
- instructions

![Prescription template form screen](/img/screenshots/prescription-template-form-screen.jpg)

## Why templates matter

Templates save time during consultations and help clinics standardize common treatment plans for repeated conditions.
