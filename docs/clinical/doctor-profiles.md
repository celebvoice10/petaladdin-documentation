# Doctor Profiles

The `Doctor Profiles` module helps clinics maintain clinical staff records for doctors. These profiles are created from staff members whose staff type is `Doctor`.

## What users can do here

- review doctor profiles
- search by doctor name, registration, email, or phone
- filter by status
- open doctor records
- edit doctor details when permitted
- use `Add Doctor Staff`

## Main actions

The page can show:

- `Add Doctor Staff`
- `Filters`
- `Reset`

## What the doctor list shows

Each row can include:

- doctor name
- qualification
- email
- phone
- clinic
- registration number
- status

## Why this page matters

Doctor profiles improve:

- consultation clarity
- staff identification
- certificate generation
- clinic reporting

## Best practice

Keep doctor registration and contact details current. Accurate doctor records make consultation and certificate workflows more reliable.
