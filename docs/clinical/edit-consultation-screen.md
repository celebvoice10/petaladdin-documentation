# Edit Consultation Screen

The `Edit Consultation` screen uses the same multi-step layout as the create form, but it updates an existing clinical record.

## When to use this screen

Open it when:

- notes need correction
- vitals were missed
- diagnosis or advice must be refined
- prescription items must be updated
- follow-up information must be added

## Important note

Completed consultations should be reviewed carefully before editing, especially if PDFs, billing, vaccination, or other downstream workflows already depend on the record.
