# Deworming

The `Deworming` module helps clinics schedule, complete, and review deworming records for pets in the active clinic.

## What users can do here

- search deworming records by dewormer, pet, parent, or phone
- filter by status
- sort by due date or dewormer name
- use `Add Deworming`
- review previous deworming history
- use AutoFill for planned future dates

## Deworming list screen

The list includes:

- search
- advanced filters
- status filters
- sorting
- quick access to `Add Deworming`

This page is useful for finding:

- due records
- overdue records
- completed deworming entries

## Deworming plan workflow

The deworming form is split into two steps:

### 1. Deworming Context

In this step, users:

- search and select the pet parent
- select the pet
- optionally select a consultation
- add notes

### 2. Deworming Plan

In this step, users add one or more rows with:

- dewormer name
- dose date
- status

## AutoFill support

The deworming form includes an `AutoFill` option.

This helps create future schedule rows based on:

- the pet's date of birth
- the most recent deworming date available

AutoFill is useful for quickly preparing a preventive-care schedule, but the dates should still be reviewed before saving.

## Previous history panel

The history section shows recent deworming records for the selected pet, which helps users:

- review the last administered date
- avoid duplicated entries
- compare the new plan against recent care

## Common mistakes to avoid

- using AutoFill before selecting the correct pet
- leaving old draft rows in place after plan changes
- saving the wrong status for a completed dose

## Best practice

Use previous history as your reference point before creating new future dates. AutoFill is helpful, but staff should still confirm the plan matches the pet's real care schedule.
