# Subscription

The `Subscription` page helps clinic administrators review the current package, request renewals or upgrades, and track approval status.

## What users can do here

- review the current package
- check package expiry
- see pending approval requests
- choose a package
- submit a cash subscription request
- print approved subscription receipts
- review subscription history

## What the page shows

The subscription screen can include:

- current package name
- active-until date
- clinic limit
- staff limit
- pending approval status
- package cards with monthly pricing
- subscription request form
- subscription history

## Requesting a subscription

1. Open `Subscription`.
2. Review available packages.
3. Choose a package.
4. Enter the number of months.
5. Select the start date.
6. Add an optional note.
7. Select `Request Subscription`.

## Approval flow

Requests are submitted for approval and may show:

- `Pending Approval`
- active package after approval
- printable receipt for approved subscriptions

While a request is pending, the clinic cannot submit another one.

## Renewal and expiry review

The page highlights:

- days remaining before expiry
- expired packages
- renewal timing

Clinic admins should review this page early when the package is close to expiring so staff and clinic limits are not disrupted.
