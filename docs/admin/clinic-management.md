# Clinics

The `Clinics` module helps clinic administrators manage clinic profiles, legal identifiers, contact details, and payment information for each allotted clinic.

## What users can do here

- view clinic allotment and remaining clinic slots
- search clinics by name, phone, email, or tenant admin
- filter by status
- use `Add Clinic`
- open clinic details
- edit an existing clinic profile

## Main actions

The page can show:

- `Add Clinic`
- `Filters`
- `Reset`

## What the clinic list shows

Each clinic row can include:

- clinic name
- tagline
- tenant admin
- admin email
- phone
- email
- status

## Creating or editing a clinic

The clinic form can include:

- tenant admin
- clinic name
- status
- tagline
- phone
- email
- time zone
- CIN number
- GSTIN
- drug license number
- GPay number
- bank name
- bank account number
- IFSC code
- address

## When to update clinic details

Review this page when:

- a new branch opens
- contact information changes
- tax or legal registration details change
- payment collection details need correction

## Important notes

- Clinic allotment limits come from the active subscription package.
- Tenant admins may only create clinics within their remaining slots.
- Accurate GST, banking, and address details help prevent invoice and payment problems later.
