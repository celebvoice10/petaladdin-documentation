# Suppliers

The `Suppliers` module helps clinics manage vendors for medicines, food, and stock-related purchasing.

## What users can do here

- use `Add Supplier`
- edit supplier details
- review supplier status
- check clinic mapping
- review GST and contact information

## What the supplier list shows

Each supplier record can show:

- supplier name
- contact person
- phone number
- clinic
- GST number
- status

## When to use this page

Use the `Suppliers` page when:

- a new vendor is added
- supplier contact details change
- tax details need correction
- purchasing staff need the correct supplier for a purchase
