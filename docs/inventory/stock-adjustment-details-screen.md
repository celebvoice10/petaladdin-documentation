# Stock Adjustment Details Screen

The stock adjustment details page is used to review a saved manual stock correction.

## What users should expect here

This screen usually acts as the audit view for an adjustment and should be reviewed for:

- adjustment number
- clinic
- type
- reason
- date
- notes
- affected products
- quantities
- batch references
- creator or audit context

## When to use this screen

Open it when:

- investigating missing stock
- reviewing expiry removal entries
- confirming an opening-balance correction
- checking why stock moved without a purchase or sale
