# Categories

The `Product Categories` module groups products, medicines, food, and services for inventory organization and billing logic.

## What users can do here

- use `Add Category`
- edit a category
- review category status
- see how many products belong to each category

## What the category list shows

The list can include:

- category name
- clinic
- product count
- status

## Why categories matter

Categories support:

- cleaner product organization
- inventory filtering
- billing setup
- discount and membership rules
