# Pet Profile Workspace

The `Pet Profile` page is a combined history workspace. It lets users select a pet parent and pet, then review clinical and billing history in one place.

## Who uses this page

- doctors
- nurses
- front-desk staff
- clinic administrators reviewing patient history

## What the workspace includes

After selecting a pet parent and pet, the page can show:

- pet parent details
- selected pet details
- vaccination history
- deworming history
- prescription history
- bills and invoices

## How to load a profile

1. Open `Pet Profile`.
2. Search for the pet parent by name, phone, or email.
3. Choose the correct pet parent from live search results.
4. Select the pet from the pet dropdown.
5. Review the loaded profile sections.

## Pet Parent Details panel

This section can include:

- name
- phone
- email
- address
- membership plan
- membership expiry
- number of pets

## Selected Pet Details panel

This section can include:

- name
- species
- breed
- sex
- date of birth
- blood group
- microchip number
- kennel number
- color

## History sections

### Vaccination History

Shows recent vaccination records, due dates, administered dates, and status.

### Deworming History

Shows recent deworming records, due dates, administered dates, and status.

### Prescriptions

Shows a consultation timeline with:

- consultation date
- doctor
- diagnosis
- medicines
- dose
- route
- duration
- instructions

### Bills / Invoices

Shows financial history such as:

- invoice number
- invoice date
- linked consultation date
- total amount
- payment status

## Best use cases

Use this page when you need one combined view before:

- starting a consultation
- checking prior prescriptions
- confirming vaccination history
- reviewing outstanding invoice history during follow-up

## Tips

- always confirm the selected pet before reviewing history
- use this workspace as a review page, then open the detailed module if changes are needed
- check invoice status when clients ask about prior bills
